§code_field: Ident

Identifier for variable used for formatted code, e.g. ___code_0. Enables separation of formatting and diagnostic emission so that set_arg calls can happen in-between..

§code_init: TokenStream

Initialization logic for code_field’s variable, e.g. let __formatted_code = /* whatever */;
