In contrast (w.r.t. struct) downcast on enum works as follows. Given * a “place” x : T[i..] * a “variant” of type forall z..,(y:t...) => E[j...] We want downcast to return a vector of types and an assertion by 1. Instantiate the type to fresh names z'... to get (y:t'...) => T[j'...] 2. Unpack the fields using y:t'... 3. Assert the constraint i == j'...
