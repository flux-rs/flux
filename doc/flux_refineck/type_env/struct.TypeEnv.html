When checking a borrow in the right hand side of an assignment x = &'?n p, we use the annotated region '?n in the type of the result. This region will only be used temporarily and then replaced by the region in the type of the x after the assignment.

pub(crate) fn ptr_to_borrow( &mut self, rcx: &mut RefineCtxt<'_>, gen: &mut ConstrGen<'_, '_, '_>, place: &Place ) -> Result<(), CheckerErrKind>

Converts a pointer ptr(mut, l) to a borrow &mut T for a type T that needs to be inferred. For example, given the environment

x: i32[a], p: ptr(mut, x)

Converting the pointer to a borrow will produce

x: †i32{v: $k(v)}, p: &mut i32{v: $k(v)}

together with a constraint

i32[a] <: i32{v: $k(v)}
