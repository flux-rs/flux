struct PointsToAnalysis<'a> {
    fn_sig: Option<EarlyBinder<PolyFnSig>>,
    map: &'a Map,
}
Expand description

This implement a points to analysis for mutable references over a [FlatSet]. The analysis is a may analysis. If you want to know if a reference definitiely points to a location you have to combine it with the result of a definitely initialized analysis. See module level documentation for more details.
