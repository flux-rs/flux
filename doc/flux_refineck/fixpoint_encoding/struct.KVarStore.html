pub fn fresh( &mut self, binders: &[List<Sort>], scope: &Scope, encoding: KVarEncoding ) -> Expr

Generate a fresh kvar under several layers of binders. The variables bound in the last layer (last element of the binders slice) are used as the self arguments. The rest of the binders are appended to the scope.

Note that the returned expression will have escaping variables and it is up to the caller to put it under an appropriate number of binders.
