BaseTy(BaseTy)

As a base type bty without any refinements is equivalent to bty{vs : true} we don’t technically need this variant, but we keep it around to simplify desugaring.

Constr(Expr, Box<Ty>)

Constrained types {T | p} are like existentials but without binders, and are useful for specifying constraints on indexed values e.g. {i32[@a] | 0 <= a}
