Expand description

Environment used to map from [rty::Var] to a fixpoint::LocalVar.

Fields§

§local_var_gen: IndexGen<LocalVar>§fvars: UnordMap<Name, LocalVar>§layers: Vec<Vec<LocalVar>>

Layers of late bound variables

§reverse_map: UnordMap<LocalVar, Expr>

While it might seem like the signature should be UnordMap<fixpoint::LocalVar, rty::Var>, we encode the arguments to kvars (which can be arbitrary expressions) as local variables; thus we need to keep the output as an [rty::Expr] to reflect this.
