fn parse_atom(cx: &mut ParseCtxt<'_>, allow_struct: bool) -> ParseResult<Expr>
Expand description
⟨atom⟩ := ⟨if_expr⟩
        | ⟨lit⟩
        | ( ⟨expr⟩ )
        | ( ⟨expr⟩,* )
        | ⟨epath⟩
        | ⟨bounded_quant⟩
        |  <⟨ty⟩ as ⟨path⟩> :: ⟨ident⟩
        | [binop]
        | ⟨epath⟩ { ⟨constructor_arg⟩,* }    if allow_struct
        | { ⟨constructor_arg⟩,* }            if allow_struct
        | #{ ⟨expr⟩,* }
